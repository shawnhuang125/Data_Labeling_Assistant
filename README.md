readme_content = """# 餐廳評論資料標註工具 (Review JSON Editor)

這是一個基於 Python tkinter 開發的 GUI 應用程式，專門用於檢視、編輯與標註餐廳評論資料集（JSON 格式）。

特別針對「同一店家多筆評論」的情境進行了優化，包含自動填入與同步更新功能，大幅提升資料清洗與標籤標註的效率。

## 功能列表

### 介面與操作
* **深色模式 (Dark Theme)**：長時間工作不傷眼。
* **響應式佈局**：視窗縮放時，評論內容區塊會自動調整大小。
* **自定義多選選單 (Multi-Select Dropdown)**：支援多選勾選（如：同時選擇「麵食」與「炸物」）。彈出視窗寬度自動對齊，操作直覺。即使資料不在預設選項內（自定義標籤），也能正確顯示與保存。

### 智慧資料處理 (核心功能)
* **同店家標籤同步 (Batch Update)**：當您修改某筆評論的店家共用屬性（如 Food Type, Cuisine, Tags, Atmosphere, Flavor）時，系統會自動搜尋檔案中所有 original_id 相同的評論，並同步更新這些欄位。
    * 優點：不用一筆一筆重複修改同一家店的分類。
* **自動填入 (Auto-fill)**：點選某一筆資料時，若該欄位為空，系統會自動嘗試從快取 (Cache) 中填入該店家已知的分類資訊。
* **自動暫存 (Auto-save on Switch)**：切換到下一筆資料時，會自動將上一筆的變更寫入記憶體，防止手滑忘記按套用。

### 檔案操作
* **載入 JSON**：讀取標準 JSON 陣列格式。
* **儲存 (快速存檔)**：直接覆蓋目前開啟的檔案，不跳出對話框，方便隨時保存進度。
* **另存新檔**：將編輯後的資料另存為新檔案。

### 編輯功能
* **新增此店家評論**：快速在目前選取的位置下方，新增一筆屬於同一家店（繼承 ID 與 Name）的空白評論。
* **刪除選取項目**：刪除不需要的評論資料。

## 資料欄位說明

此工具主要編輯以下欄位：

| 欄位鍵值 (Key) | 顯示名稱 | 類型 | 說明 | 同步更新 |
| :--- | :--- | :--- | :--- | :--- |
| original_id | 原始編號 | 字串 | 資料庫唯一識別碼 (唯讀)。 | |
| name | 店名 | 字串 | 店家名稱 (唯讀)。 | 否 |
| review_text | 評論內容 | 字串 | 原始評論文字。 | 否 |
| review_summary | 評論摘要 | 字串 | 評論的重點摘要。 | 否 |
| level | 等級 | 數字/字串 | 1=負評, 2=普通, 3=好評。 | 否 |
| food_type | 食物類型 | 列表 | e.g., ["麵食", "炸物"] | 是 |
| cuisine_type | 料理菜系 | 列表 | e.g., ["日式料理"] | 是 |
| service_tags | 服務標籤 | 列表 | e.g., ["有WIFI", "可刷卡"] | 是 |
| atmosphere | 氣氛 | 列表 | e.g., ["安靜", "浪漫"] | 是 |
| flavor | 口味描述 | 列表 | e.g., ["酸", "辣"] | 是 |

**注意**：「同步更新」欄位代表只要修改其中一筆，該 ID 下的所有評論都會被一起修改。
